# Instagram
It is a fully functional Instagram clone made in Django (In progress)

## Backend Stuff Done:

*Profile*
* login/signup page
* basic user profile page
* edit profile
* password change

*Posts page*
* Profile Search
* Single Post Page
* Dynamic Like, Comment, Save 
* Share post in Messages
* Explore Page
* Saved Post Page

*Message System*
* Inbox Page
* Create new inboxes
* Dynamic messages send/recieve

*Notification*
* Dynamic message notification
* Like, Comment, follow notification

## Frontend Stuff Done:

* *Navigation-bar*
* *Notification Panel*
* *Single Post View*
